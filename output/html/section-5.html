Section 2.1 Topological catch-all

Definition 2.1.1. Separation axioms.

  • A topological space \(\X\) is \(T_1\) if whenever \(x \neq y\text{,}\) there exists an open set containing \(y\) but not \(x\text{.}\)

  • A topological space \(\X\) is called Hausdorff if it is \(T_2\text{;}\) that is, if \(x \neq y\) then there are disjoint open sets \(U, V\) with \(x \in U, y \in V\text{.}\)

  • A topological space \(\X\) is called normal if it is \(T_4\text{;}\) that is, \(\X\) is \(T_1\) and for any disjoint closed sets \(A, B\) in \(\X\text{,}\) there are disjoint open sets \(U, V\) with \(A \subset U\) and \(B \subset V\text{.}\)

Definition 2.1.2. Continuous functions.

Suppose that \(\X, \Y\) are topological spaces.

  • A function is called continuous if \(f\inv(V)\) is open for every open set \(V \subset \X\text{.}\)

  • A function is called continuous at \(x\) if \(f\inv(V)\) is open for every neighborhood \(V\) of \(x\text{.}\)

  • A bijective function \(f:\X \to \Y\) is called a homeomorphism if \(f, f\inv\) are continuous.

We'll use the notation \(C(\X, \Y)\) to designate the family of continuous functions \(f: \X \to \Y\text{.}\)

We'll need a couple of extension theorems. They have names, so you know they must be important.

Theorem 2.1.3. Uryshon's Lemma.

Let \(X\) be a normal space. If \(A, B\) are disjoint closed sets in \(X\text{,}\) then there exists \(f \in C(X, [0,1])\) such that \(f = 0\) on \(A\) and \(f = 1\) on \(>B\text{.}\)

Theorem 2.1.4.

Let \(X\) be a normal space. If \(A\) is a closed subset of \(\X\) and \(f \in C(A, [a,b])\text{,}\) there exists a continuous extension \(F \in C(X, [a,b])\) so that the restriction \(F\vert_A\) is \(f\text{.}\)
