Section 2.2 Nets

In general topological spaces, we usually can't get away with thinking about sequences. Instead, we'll capture convergence in topological spaces with nets. A sequence is a function \(x: \mathbb{N} \to \mathcal{X}\) that maps the natural numbers into a space \(\mathcal{X}\text{.}\) We want to think of nets as functions that map more general index sets into a space. We need to retain a notion of forward progress to address convergence, which we get with the notion of directed sets.

Definition 2.2.1.

A directed set is a set \(A\) with a binary relation \(\preceq\) with the properties that

  1. (self-relation): \(\alpha \preceq \alpha\) for all \(\alpha \in \mathcal{A}\text{;}\)

  2. (transitivity): if \(\alpha \preceq \beta\) and \(\beta \preceq \gamma\) then \(\alpha \preceq \gamma\text{;}\)

  3. (upper bounds): for any \(\alpha, \beta \in \mathcal{A}\text{,}\) there exists \(\gamma \in \mathcal{A}\) such that \(\alpha \preceq \gamma\) and \(\beta \preceq \gamma\text{.}\)

Not every pair of elements in a directed set relate (unlike a totally ordered set), but we do have the upper bound property that will allow us to define convergence.

Definition 2.2.2.

A net in \(\mathcal{X}\) is a function \(x\) from a directed set \(\mathcal{A}\) into a space \(\mathcal{X}\text{.}\) We usually write \(x(\alpha) = x_\alpha\) and write the net as \(\langle x_\alpha \rangle_{\alpha \in \mathcal A} = \langle x_\alpha \rangle\text{.}\)

Definition 2.2.3.

Now let \(\mathcal{X}\) be a topological space and \(E \subset \mathcal{X}\text{.}\)

  • A net \(\net{x_\alpha}\) is eventually in \(E\) if there exists \(\alpha_0 \in \mathcal{A}\) such \(x_\alpha \in \mathcal{A}\) for all \(\alpha \succeq \alpha_0\text{.}\)

  • A net \(\net{x_\alpha}\) is frequently in \(E\) if for every \(\alpha \in \mathcal{A}\) there exists \(\beta \succeq \alpha\) such that \(x_\beta \in E\text{.}\)

  • A point \(x \in \mathcal{X}\) is a limit of \(\net{x_\alpha}\) if for every neighborhood \(U\) of \(x\text{,}\) \(\net{x_\alpha}\) is eventually in \(U\text{.}\) In this case, we say that \(x_\alpha\) converges to \(x\) and we write \(x_\alpha \to x\text{.}\)

  • A point \(x \in E\) is a cluster point of \(\net{x_\alpha}\) if for every neighborhood \(U\) if \(x\text{,}\) \(\net{x_\alpha}\) is frequently in \(E\text{.}\)

Now, we'll show that nets have many of the desirable properties that sequences do.

Proposition 2.2.4.

If \(\mathcal X\) is a topological space, \(E \subset \mathcal{X}\text{,}\) and \(x \in \mathcal X\text{,}\) then \(x\) is an accumulation point of \(E\) if and only if there exists a net in \(E\backslash\{x\}\) that converges to \(x\text{.}\) Likewise, \(x \in \cc{E}\) if and only if there is a net in \(E\) that converges to \(x\text{.}\)

Proof.

If \(x\) is a limit point of \(E\text{,}\) then let \(\N\) be the set of neighborhoods of \(x\text{,}\) directed by reverse inclusion. For each \(U \in \N\text{,}\) choose an element \(x_U \in (U\backslash\{x\})\cap E\text{.}\) Then by definition, \(x_U \to x\text{.}\)

On the other hand, if \(x_\alpha\) is a net in \(E\backslash\{x\}\) and \(x_\alpha \to x\text{,}\) then for any neighborhood \(U \in \N\text{,}\) the punctured set \(U\backslash\{x\}\) contains some point in \(x_\alpha\text{.}\) Thus, \(x\) is a limit point of \(E\text{.}\)

The second statement follows from the definition and the observation that the closure of a set contains its limit points.

We also get a net definition of continuity akin to the typical sequence definition.

Proposition 2.2.5.

Let \(\mathcal X, \mathcal Y\) be topological spaces and \(f: \mathcal X \to \mathcal Y\text{.}\) The function \(f\) is continuous at \(x \in \mathcal X\) if and only if for every net \(\net{x_\alpha}\) converging to \(x\text{,}\) the net \(\net{f(x_\alpha)}\) converges to \(f(x)\text{.}\)

Proof.

Suppose that \(f\) is continuous at \(x\text{.}\) Then for a neighborhood \(V\) of \(f(x)\text{,}\) \(f\inv(V)\) is a neighborhood of \(x\text{.}\) Now suppose that a net \(x_\alpha\) converges to \(x\text{.}\) By definition, \(x_\alpha\) must eventually be in \(f\inv(V)\text{,}\) which implies that \(\net{f(x_\alpha)}\) is eventually in \(V\text{.}\) But this means that \(f(x_\alpha) \to f(x)\text{,}\) as the choice of \(V\) was arbitrary.

Finally, we'll state a net version of the theorem for sequences that states that sequential cluster points have convergent subsequences - be careful with this one, because a subnet it isn't quite the direct analogy that it appears to be.

Subnets.

There are at least three non-equivalent definitions of subnet. Despite the complications, the idea is generally the same - to recover as many theorems involving convergent sequences as possible. To see more explanation, see the article on subnets 1  on Wikipedia.

Definition 2.2.6.

A subnet of a net \(\net{x_\alpha}_{\alpha \in \mathcal A}\) is a net \(\net{y_\beta}_{\beta \in \mathcal B}\) and a map \(\beta \mapsto \alpha_\beta\) from \(\mathcal B \to \mathcal A\) such that the following hold:

  • For every \(\alpha_0 \in \mathcal A\) there exists \(\beta_0 \in \mathcal B\) such that \(\alpha_\beta \succeq \alpha_0\) whenever \(\beta \succeq \beta_0\text{.}\)

  • \(\displaystyle y_\beta = x_{\alpha_\beta}\)

If \(\net{x_\alpha}\) converges to a point \(x\text{,}\) then so too must any subnet \(\net{x_{\alpha_\beta}}\)

Proposition 2.2.7.

If \(\net{x_\alpha}\) is a net in \(\mathcal X\) then \(x\) is a cluster point of \(x_\alpha\) if and only if there is a subnet of \(x_\alpha\) converging to \(x\text{.}\)

Proof.

Suppose that \(\net{x_\alpha}\) has a subnet \(\net{y_\beta} = \net{x_{\alpha_\beta}}\) converging to \(x\text{.}\) Let \(U\) be a neighborhood of \(x\text{.}\) Now choose \(\beta_1\) so that \(y_{\beta} \in U\) whenever \({\beta} \succeq \beta_1\) (which we can do by convergence). For a given \(\alpha\text{,}\) choose \(\beta_2\) so that \(\alpha_{\beta} \succeq \alpha\) for \(\beta \succeq \beta_2\) (which we can do by the definition of a subnet). Since \(\mathcal B\) is a directed set, choose \(\beta \in \mathcal B\) so that \(\beta \succeq \beta_1\) and \(\beta \succeq \beta_2\text{.}\) Then \(\alpha_\beta \succeq \alpha\) and \(x_{\alpha_\beta} = y_\beta \in U\text{.}\) That is, \(x_{\alpha}\) is frequently in \(U\text{,}\) and so \(x\) is a cluster point of \(\net{x_\alpha}\text{.}\)

In the other direction, if \(x\) is a cluster point of \(\net{x_\alpha}\text{,}\) the idea will be to use the neighborhoods at \(x\) as the index. Let \(\mathcal N\) be the neighborhoods of \(x\text{.}\) Construct the product of directed sets \(\mathcal N \times \mathcal A\) with the relation \((U, \alpha) \preceq (U_1, \alpha_1)\) when \(U \supset U_1\) and \(\alpha \preceq \alpha_1\text{.}\) Then using the upper bound property, for each \((U, \beta) \in \mathcal N \times \mathcal A\text{,}\) we can choose \(\alpha_{(U, \beta)}\) such that \(\alpha_{(U, \beta)} \succeq \beta\) and \(x_{\alpha_{(U, \beta)}} \in U\text{.}\) If \({(U', \beta')} \succeq (U, \beta)\text{,}\) we have

\begin{equation*} \alpha_{{(U', \beta')}} \succeq {\beta'} \succeq \beta \end{equation*}

and

\begin{equation*} x_{\alpha_{{(U', \beta')}}} \in {U'} \subset U\text{.} \end{equation*}

Thus \(\net{x_{\alpha_{{(U, \beta)}}}}\) is a subnet of \(\net{x_\alpha}\) that converges to \(x\text{.}\)
