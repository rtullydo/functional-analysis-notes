If \(T\) is the zero operator, then we're done. If \(T \neq 0\text{,}\) there exists a sequence of unit vectors \(h_n\) so that

\begin{equation*} \abs{\ip{T h_n}{h_n}} \to \norm{T}. \end{equation*}

A limit point argument produces a subsequence \(h_n\) so that

\begin{equation*} \ip{T h_n}{h_n} \to \la \end{equation*}

where \(\abs{\la} = \norm{T}\text{.}\)

Since \(\abs{\la} = \norm{T}\text{,}\) we get

\begin{align*} 0 \amp \leq \norm{(T - \la)h_n}^2\\ \amp= \norm{Th}^2 - 2\la\ip{Th_n}{h_n} + \la^2\\ \amp= 2\la^2 - 2 \la\ip{Th_n}{h_n}. \end{align*}

But \(\ip{Th_n}{h_n} \to \la\text{,}\) so the computation above implies that \(\norm{(T - \la)h_n} \to 0\text{.}\) Applying PropositionÂ 1.2.12, we conclude that \(\la \in \sigma_p(T)\text{.}\)
