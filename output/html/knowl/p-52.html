If \(T\) is the zero operator, then we're done. If \(T \neq 0\text{,}\) there exists a sequence of unit vectors \(h_n\) so that

\begin{equation*} \abs{\ip{T h_n}{h_n}} \to \norm{T}. \end{equation*}

A limit point argument produces a subsequence \(h_n\) so that

\begin{equation*} \ip{T h_n}{h_n} \to \la \end{equation*}

where \(\abs{\la} = \norm{T}\text{.}\)
