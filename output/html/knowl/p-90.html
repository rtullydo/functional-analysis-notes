A topological vector space (TVS) is a vector space \(\mathcal{X}\) over a field \(\mathbb{F}\) and a topology \(\mathcal{T}\) so that

  1. addition is continuous - that is, the map \((x, y)\mapsto x + y\) is continuous.

  2. scalar multiplication is continuous - that is, the map \((\alpha, x)\mapsto \alpha x\) is continuous.
