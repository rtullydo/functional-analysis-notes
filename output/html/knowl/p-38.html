The hypotheses imply that there exists a sequence of unit vectors \(h_n\) so that \(\norm{(T - \la)h_n} \to 0\text{.}\) By compactness, this sequence has a subsequence \(h_{n_k}\) and a vector \(f\) so that \(\norm{Th_{n_k} - f}\to 0\text{.}\) Combining these two facts, we can use an algebra trick to get

\begin{align*} h_{n_k} = \la\inv \la h_{n_k} \amp= \la\inv \la(h_{n_k} - \la\inv T h_{n_k} + \la \inv T h_{n_k})\\ \amp=\la\inv [(\la - T) h_{n_k}) + T h_{n_k}] \\ \amp\to \la\inv[0 + f] = \la\inv f. \end{align*}

Since the \(h_n\) are unit vectors, so is any limit point of the sequence, and so \(1 = \norm{\la\inv f} = \abs{\la}\inv \norm{f}\text{,}\) and so \(\norm{f} \neq 0\text{.}\) Furthermore, \(T h_{n_k} \to \la\inv Tf\text{.}\) Since we also have \(T h_{n_k} \to f\text{,}\) it must be that \(f = \la\inv T f\text{,}\) or rather that \(T f = \la f\text{.}\) That is, \(f\neq 0\text{,}\) \(f \in \ker(T - \la)\text{,}\) which gives that \(\la \in \sigma_p(T)\text{.}\)
