Since \(A\) is normal, it is easy to see that so must be \(A - \la\text{.}\) By [provisional cross-reference: prop-normaladnorm], we have that \(\norm{(A - \la)h} = \norm{(A - \la)\ad h}\) for all \(h\text{.}\) But then \(h \in \ker(A - \ad)\) if and only if \(h \in \ker(A - \la)\ad\text{.}\)

If \(h \in \ker(A - \la)\text{,}\) then \(Ah = \la h \in \ker(A - \la)\) and \(A\ad h = \cc\la h \in \ker(A - \la)\text{.}\) We conclude that \(\ker(A - \la)\) is a reducing subspace for \(A\text{.}\)
