  • A topological space \(\X\) is \(T_1\) if whenever \(x \neq y\text{,}\) there exists an open set containing \(y\) but not \(x\text{.}\)

  • A topological space \(\X\) is called Hausdorff if it is \(T_2\text{;}\) that is, if \(x \neq y\) then there are disjoint open sets \(U, V\) with \(x \in U, y \in V\text{.}\)

  • A topological space \(\X\) is called normal if it is \(T_4\text{;}\) that is, \(\X\) is \(T_1\) and for any disjoint closed sets \(A, B\) in \(\X\text{,}\) there are disjoint open sets \(U, V\) with \(A \subset U\) and \(B \subset V\text{.}\)
