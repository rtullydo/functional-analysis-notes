Item.

\(\X\) is compact if every open cover of \(\X\) has a finite subcover. That is, given a collection of open sets so that \(\X = \bigcup_{\alpha \in \mathcal A} U_\alpha\text{,}\) there is a finite subset \(\mathcal B\) of \(\mathcal A\) so that \(X = \bigcup_{\alpha \in \mathcal B} U_\alpha\text{.}\)
