Proof

First, note that \((3) \Rightarrow (1)\) follows from PropositionÂ 1.3.4.

We will show that \((1) \Rightarrow (3) \Rightarrow (2) \Rightarrow (1)\text{.}\)

\((1) \Rightarrow (3)\text{:}\) We use the fact that since \(T\) is compact, the closure of \(T(\ball \hilbert)\) contains a countable dense subset (one can construct this set using the fact that the image of the ball under \(T\) is totally bounded). Then \(\mathcal L = \cl \ran T\) is a separable subspace of \(\K\text{.}\) Let \(\{e_j\}\) be a basis for \(\mathcal L\text{,}\) and let \(P_n\) be the projection of \(\K\) onto the closed linear span of the first \(n\) basis vectors; that is, \(P_n\) is the projection onto \(\vee\{e_j: 1 \leq j \leq n\}\text{.}\) Now set \(T_n = P_n T\text{.}\) Each \(T_n\) is a finite rank operator.

We will show that \(\norm{T_n - T} \to 0\) by establishing a uniform bound on \(\norm{T_n h - T h}\) for \(\norm{h}\leq 1\text{.}\) Choose \(\eps > 0\text{.}\) By compactness, we can find a finite collection of vectors \(h_1, \ldots, h_m\) so that

\begin{equation*} T(\ball \hilbert) \subset \bigcup_{j = 1}^m B(Th_j, \ep/3). \end{equation*}

For \(h \in \ball \hilbert\text{,}\) choose \(h_j\) so that \(\norm{Th - Th_j} \leq \eps/3\text{.}\) Then for any \(n \in \N\text{,}\)

\begin{align*} \norm{T_n h - T h} \amp\leq \norm{T_n h - T_n h_j } + \norm{T_n h_j - T h_j} + \norm{T h_j - T h} \\ \amp \leq\norm{P_n(T h - T h_j) } + \norm{T_n h_j - T h_j} + \norm{T h_j - T h} \\ \amp\leq 2\norm{T h_j - T h} + \norm{T_n h_j - T h_j}\\ \amp\leq \frac{2\eps}{3} + \norm{T_n h_j - T h_j}. \end{align*}

Now, we show that we can find \(n_0\) so that \(\norm{T_n h_j - T h_j} \leq \eps/3\text{.}\) Certainly \(Th_j \in \mathcal{L}\text{,}\) and so can be expanded as \(Th_j = \sum_{k=1}^\infty \ip{Th_j}{ e_k} e_k\text{.}\) On the other hand,

\begin{equation*} T_n h_j = P_n T h_j = \sum_{k=1}^n \ip{Th_j}{ e_k} e_k \end{equation*}

Together, this gives \(\norm{T_n h_j - T h_j} \to 0\text{,}\) and so the desired \(n_0\) exists. Since \(h\) was an arbitrary vector with \(\norm{h} \leq 1\text{,}\) this bound is uniform, and so in the operator norm we have \(\norm{T_n - T} \to 0\text{.}\)

\(3 \Rightarrow 2:\) Suppose that \(T_n\) is a sequence in \(\BOP_{00}(\hilbert, \K)\) with \(\norm{T_n - T} \to 0\text{.}\) Then \(\norm{T_n\ad - T\ad} = \norm{T_n - T} \to 0\text{.}\) But \(T_n\ad\) is also in \(\BOP_{00}(\hilbert, \K)\) (Exercise). Since \(3 \Rightarrow 1\text{,}\) \(T\ad\) is compact.

\(2 \Rightarrow 1\text{:}\) (Exercise)
