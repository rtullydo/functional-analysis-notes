Definition 2.3.1.

Let \(\X\) be a topological space.

  • \(\X\) is compact if every open cover of \(\X\) has a finite subcover. That is, given a collection of open sets so that \(\X = \bigcup_{\alpha \in \mathcal A} U_\alpha\text{,}\) there is a finite subset \(\mathcal B\) of \(\mathcal A\) so that \(X = \bigcup_{\alpha \in \mathcal B} U_\alpha\text{.}\)

  • A subset \(Y\) of a topological space is compact if it is compact in the relative topology. That is, \(Y \subset \X\) is compact if and only if for every collection of open sets in \(\X\) with \(Y \subset \bigcup_{\alpha \in \mathcal A}\text{,}\) there is a finite subset \(\mathcal B\) of \(\mathcal A\) so that \(Y \subset \bigcup_{\alpha \in \mathcal B} U_\alpha\text{.}\)

  • A set \(Y \subset \X\) is called precompact if its closure is compact.
