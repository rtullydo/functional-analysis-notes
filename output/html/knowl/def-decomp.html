Definition 1.2.3.

If \(\M_i\) is a collection of pairwise orthogonal subspaces of \(\hilbert\text{,}\) then set

\begin{equation*} \bigoplus_{i} \M_i = \bigvee_i \M_i. \end{equation*}

If \(\M, \N\) are closed linear subspaces of \(\hilbert\text{,}\) then set the orthogonal difference of \(\M\) and \(\N\) to be

\begin{equation*} \M \ominus \N = \M \cap \N^\perp. \end{equation*}
