Definition 2.1.3.

A seminorm is a real-valued function \(p:\mathcal{X} \to \R\) satisfying the following properties:

  1. Triangle inequality:
    \begin{equation*} \forall x, y \in \mathcal{X}, p(x + y) \leq p(x) + p(y) \end{equation*}
  2. Homogeneity:
    \begin{equation*} \forall \alpha \in \mathbb{F}, x \in \mathcal{X}, p(\alpha x) = \abs{\alpha}p(x) \end{equation*}
  3. Non-negativity:
    \begin{equation*} p(x) \geq 0 \,\,\,\forall x \in \mathcal{X} \end{equation*}
